Instalação das Ferramentas

Esta página concentra o setup das 3 ferramentas de segurança com os comandos já documentados nas páginas de cada ferramenta. A proposta é reduzir o tempo de configuração e padronizar o início do trabalho de novos desenvolvedores.

Windows ou outras distribuições do linux

Os comandos abaixo são para Linux/macOS. Se você está no Windows, utilize o WSL 2 (Windows Subsystem for Linux) ou adapte os comandos conforme a documentação oficial de cada ferramenta.

1. Script de Configuração de Ambiente

Para agilizar o onboarding, disponibilizamos um script único com os mesmos passos de instalação desta página.

Arquivo disponível: script.sh

Baixar script.sh
Bash
chmod +x script.sh
./script.sh

2. Instalação das Ferramentas

2.1 Trivy

Ferramenta de SCA para detectar vulnerabilidades em dependências e arquivos de projeto.

Bash
sudo apt-get install trivy

2.2 Semgrep

Ferramenta de SAST para análise estática de padrões inseguros no código.

Bash
sudo apt update
sudo apt install -y pipx
pipx ensurepath
pipx install semgrep
Autenticação Obrigatória Semgrep Pro Engine

Semgrep requer login para usar a Pro Engine e as regras avançadas. Execute:

Bash
semgrep login

Se preferir usar token (útil em CI), veja a página detalhada do Semgrep.

2.3 Gitleaks

Ferramenta de Secrets Detection para identificar vazamento de credenciais no repositório.

Bash
wget https://github.com/gitleaks/gitleaks/releases/download/v8.18.4/gitleaks_8.18.4_linux_x64.tar.gz
tar -xf gitleaks_8.18.4_linux_x64.tar.gz
sudo mv gitleaks /usr/local/bin/
gitleaks version

3. Validação Rápida

Bash
trivy --version
semgrep --version
gitleaks version

Se os 3 comandos retornarem versão, o ambiente está pronto para execução automatizada.
